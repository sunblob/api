# Coffee API

> Backend API для информационной системы мониторинга пунктов реализации различной продукции.

## Установка зависимостей

```
npm install
```

## Запуск приложения

```
# запуск в режиме разработки
npm run dev

# запуск в режиме выпуска
npm start
```
